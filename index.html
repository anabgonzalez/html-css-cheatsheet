HTML & CSS CHEATSHEET

HTML Elements

Tag Name Description
<p> Paragraph The <p> HTML element represents a paragraph.
<a> Anchor The <a> HTML element (or anchor element), with its href attribute, creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address.
<br> Line Break The <br> HTML element produces a line break in text (carriage-return).
<div> Content Division The <div> HTML element is the generic container for flow content.
